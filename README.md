# IA-HELP
IA HELP é um repositório de websites e jogos simples desenvolvidos com apoio de Inteligência Artificial, focado em aprendizagem prática, prototipagem rápida e demonstração de competências em desenvolvimento web.  Inclui jogos clássicos e projectos interactivos como Rock Paper Scissors, Snake, Casino Roulette, Monopoly, Poker e outros.
